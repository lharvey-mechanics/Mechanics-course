Key terms

OBD-II

Standard onboard system for monitoring emissions-related performance and reporting faults.

In the shop: Connecting a scan tool to read codes and monitor data.

DLC

Diagnostic Link Connector (scan tool port).

In the shop: Locating the DLC under the dash.

MIL

Malfunction Indicator Lamp (check engine light).

In the shop: Explaining why the MIL turns on for emissions faults.

DTC

Diagnostic Trouble Code stored when a fault is detected.

In the shop: Retrieving P-codes after a MIL appears.

Pending code

A fault detected but not confirmed enough to turn on the MIL yet.

In the shop: Seeing a pending misfire code after a brief issue.

Readiness monitors

Self-tests that must complete before inspection/verification.

In the shop: Checking monitors after a battery disconnect.

Freeze frame data

Snapshot of conditions when a code set (rpm, load, temp, etc.).

In the shop: Using freeze frame to reproduce the fault conditions.

Live data

Real-time readings from sensors and calculations.

In the shop: Watching O2 sensor and fuel trim readings.

Mode $06

OBD test results for specific monitors (where supported).

In the shop: Viewing catalyst or misfire test results on some scan tools.

Drive cycle

Specific driving conditions needed to run monitors.

In the shop: Completing a drive cycle to set readiness.

Emission control system

Systems that reduce harmful exhaust emissions.

In the shop: EVAP and catalyst systems working together to reduce pollution.

EVAP system

System that captures fuel vapours and prevents them escaping to air.

In the shop: Loose gas cap causing EVAP leak code.
